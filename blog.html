Question-1: What are the different ways to select an element in the DOM?

Answer: There are many different ways to select an element in the DOM. Such as ID, Class, Tag, Query Selector, Name, Query SelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer: innerHTML allows you to manipulate the content of an element, including adding or removing HTML tags. It can be used to get or set the HTML structure within an element, while innerText shows the text exactly as rendered on the page. textContent includes all the text inside an element, even from hidden elements. It’s faster than innerText because it doesn’t have to consider CSS styles or compute layout — it just retrieves all the text in the element.

Question-3: What is event delegation in the DOM?

Answer: Event delegation is a technique in the DOM where you attach a single event listener to a parent element instead of multiple listeners to individual child elements. This works because events "bubble" up the DOM tree, meaning an event triggered on a child element will propagate up to its parent elements.

Question-4: What is event bubbling in the DOM?

Answer: Event bubbling is a concept in the DOM (Document Object Model) where an event triggered on a child element "bubbles up" through its parent elements, all the way to the top of the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer: TTo create an element in JavaScript, use document.createElement('tag'). Then, to add it to the DOM, use parent.appendChild(newElement) to append it, or parent.insertBefore(newElement, referenceElement) to insert it before another element. To remove an element, you can use parent.removeChild(element) or directly call element.remove() to remove it from the DOM. These methods allow you to dynamically manage elements on the page.
